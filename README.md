# Height Scanner for Drone  

## Project description:  

A custom height scanner system using a 2D LIDAR to map terrains using a drone.  

## Project goal:  

The objective is to:  
1. **Hardware:** Design and build a mounting system to physically attach the scanner system onto a drone.

2. **Software:** Optimize and calculate the height of the terrain the most precisely possible using a budget scanner and error correction sensors.

## Programming languages:  

Python and C
